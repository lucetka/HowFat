General concept

HowFat is a simple software for gel band quantification. It allows the user to select a rectangular area anywhere in the picture. This selected area can be assigned a name and also mass, thus setting it as standard. You can therefore estimate for example concentration of your protein prep from an SDS gel - you simply load several standards, eg. 50, 100, 150, 200, 300 and 500 ng of BSA per lane and your sample, preferably different amounts for more reliable estimation. You then make a calibration curve from the standards and quantify the sample of interest by comparing to this calibration curve. This software is also useful for comparing relative intensities of the bands, ie in homogenity estimation, PCR reaction efficiency etc.

How it works

HowFat takes every single pixel of the selected area, converts its color value to grayscale value and sums up all the pixel intensities, thus obtaining the integral intensity of the band, which, within certain mass limits, is directly proportional to the mass in the band. Band information comprising name of the band, coordinates of the selected area, user-defined mass in case of standard (or when the selected area was set to stand for background in which case the mass shoud be set to 0), and the integral intensity is then exported into a text file in such a format that it can be easily imported to a spreadsheet (eg Excel) and processed.
No correction for background is implememted in this version.

How to process your gel

  • Open the gel picture (File -> Open). ( Supported formats are bmp and jp(e)g )
  • Select bands
  • Select "Export raw band information" from the "Run!" menu
  • Import the generated file into Excel (start at row 4, delimited, semicolon as delimiter)

If you need to take background in account, you can do it manually:

  • Select an area with no bands and save it as "Standard" with 0 mass
  • average background value = integral intensity of the background / area,

    where area is (X2-X1-2)x(Y1-Y2-2).

  • corrected intensity = integral intensity - ( average background value)x(area of the band) )

However, this correction is only needed for not well destained gels or westerns with serious background.
